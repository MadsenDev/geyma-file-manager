Feature proposals July 5, 2026

What Geyma should learn next

Fourteen proposals in two tiers: the table stakes every file manager needs, and signature extras that deepen what already makes Geyma distinct — files remember, working sets, and deep customizability.

Table stakes

users will notice these missing

Expected behaviors from every serious file manager. None are glamorous; all are load-bearing.

01

Shipped

Signature extras

only Geyma could ship these

Each one grows from a mechanic that already exists — the event log, working sets, or the skin engine.
